# Cycle Map
Simple accident recording tool


### Tech
- Gulp
- Browserify
- Jade
- Babel
- Angular
- Google Maps
- Ionic
- JWT
- Socket.io


### Local Setup
1. sudo npm install -g ionic cordova
2. sudo npm install
3. gulp js
4. ionic serve
5. ionic emulate ios
